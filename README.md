# Wouterdebie Repo

## How do I install these formulae?

`brew install wouterdebie/repo/<formula>`

Or `brew tap wouterdebie/repo` and then `brew install <formula>`.

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
